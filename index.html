HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta element defines metadata such as character set, description, keywords, author, and viewpoint.
  • The script element embeds client-side scripts in the HTML document.
  • Tag refers only to what apperas inside the angle brackets.
  • The br element creates a line of empty space, or a line break, between two blocks of content.
  • src, or source, defines the location of the image file.

CSS

  • A margin indicates how much space we want around around the outside of an element.
  • A padding indicates how much space we want around the contest inside an element.
  • Declarations contain the CSS property we want to apply and the value of the property.
  • The selector defines the element or attributes to which the rules, or declarations, will apply.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: takes you from a feature branch to your main branch
  • git pull origin main: pulls down the latest main branch

JavaScript

  • A variable is named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script.
  • A function is a set of instructions that tells the computer how to preform a certain task.
  • An array is a group of related data in a variable.
  • Arrays are zero-indexed and sequential.
  • An if statement is a way of introducing decision-making into our code.
  • An index is a unique number identifier to find individual data items inside an array.
